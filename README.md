## How to Deploy
1. Create a new repository named **`your-username.github.io`**
2. Upload `index.html` and commit
3. Go to **Settings ‚Üí Pages**
4. Select **Branch: main / root**, then Save
5. Access your site at `https://your-username.github.io`
